# wxyDB
Course project for Fundamental of Database in Tsinghua University

## 基本概念
### 数据文件
    每个文件为一个数据库
    以页为单位，8192字节

### 页
    第一页（库页）
        每个数据库文件的第一页为库页，存放库的相关信息
        库信息  96字节
            表个数
    页管理位图 4096字节
    表信息  4000字节
    表首页（每个表的数据根页）位置，表名
    表位数组，表的个数4字节（末尾）

### 索引
    根级页、中间级页和叶级页组成
    B+树
        索引码值 指针
        所有结点的数据结构都相同
        最多包含n-1个索引码值K，n个指针P
        P K P K 。。。K Pn
        数据文件中的各个索引码值就必须出现且只能出现一次
        Pn将叶结点按索引码顺序链接在一起

    插入行
    分裂页
    删除行
    更新行
### 槽
    数据页上存放单条记录的空间，叫做槽
